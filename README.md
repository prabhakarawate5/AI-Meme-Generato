# AI-Powered Meme Generator

This project generates memes from text prompts using Python, PIL, and mock AI features.

## Features
- Text-to-meme generation
- Template selection
- Caption generation
- Image manipulation

## How to Run
1. Install dependencies: `pip install pillow requests`
2. Run: `python meme_generator.py`
