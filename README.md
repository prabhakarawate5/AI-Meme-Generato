ai-meme-generator
Name: AI-Powered Meme Generator
Purpose: This project creates relevant, funny memes from text prompts using Python, image manipulation, and mock AI features. It fulfills the assignment requirements for core features like text-to-meme generation, template selection, and batch output.
Tech Stack: Python 3.8+, PIL (Pillow) for image editing, and basic AI simulations (e.g., for humor detection and caption generation). Easily extensible for real AI (e.g., OpenAI API).
Status: Fully functional with generated examples uploaded to GitHub.
